# PalestraMano — Allenamento e Tracker per le Mani

Web app per l'**allenamento delle mani**: guida agli esercizi di grip, dita e polsi con contatore di ripetizioni e timer di sessione integrati. Design glassmorphism.

## Funzionalità

- **Guida esercizi** — tre esercizi mirati: Grip Strength, Estensione Dita, Rotazione Polsi
- **Contatore ripetizioni** — registra ogni rep con animazione visiva
- **Timer allenamento** — parte automaticamente al primo utilizzo
- **Scroll morbido** — navigazione fluida tra le sezioni
- **Animazioni scroll** — le card appaiono progressivamente (Intersection Observer)
- **Design glassmorphism** — pannelli in vetro smerigliato, orbi animati, font Outfit

## Come si usa

Apri `index.html` nel browser — nessun server richiesto.

1. Clicca **Inizia l'Allenamento** per avviare il timer
2. Clicca **+1 Ripetizione** dopo ogni ripetizione
3. Clicca **Reset** per azzerare tutto e fermare il timer
