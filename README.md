# PalestraMano — Hand Strength & Workout Tracker

A minimal, visually polished **hand workout web app** that guides users through grip and wrist exercises while tracking reps and elapsed time in real time.

## Live Preview

Open `index.html` directly in your browser — no build step, no server required.

## Features

- **Exercise guide** — three targeted drills: Grip Strength, Finger Extension, Wrist Rotation
- **Rep counter** — click to log each repetition with a satisfying pop animation
- **Workout timer** — auto-starts on first interaction, shows `MM:SS` elapsed time
- **Smooth scroll navigation** — anchor links glide to each section
- **Scroll-reveal cards** — exercise cards animate in via Intersection Observer
- **Glassmorphism UI** — frosted-glass panels, animated background orbs, Outfit font

## Files

| File | Description |
|---|---|
| `index.html` | Full page structure and layout |
| `style.css` | Glassmorphism styling, animations, responsive grid |

## How to Use

1. Open `index.html` in a browser
2. Click **Inizia l'Allenamento** to start the timer
3. Hit **+1 Ripetizione** after each rep
4. Hit **Reset** to clear stats and stop the timer

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- [Outfit](https://fonts.google.com/specimen/Outfit) (Google Fonts)
- Intersection Observer API
